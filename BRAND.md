# Prologis Brand Guide (for BOM Tool)

## Colors (extracted from logo)
- **Primary Dark Teal:** `#00544e` — main text, headers, nav
- **Cyan Accent:** `#00dcff` — highlights, active states, CTAs
- **Teal/Seafoam:** `#00c2b2` — secondary accents, success states
- **Lime Green:** `#7edb49` — progress indicators, positive signals
- **White:** `#ffffff` — backgrounds, cards
- **Light Gray:** `#f5f7f7` — page background, subtle borders

## Logo
- `assets/prologis-logo.png` — full color horizontal logo (1280x264 PNG)

## Typography
- Prologis uses clean sans-serif — we'll use Inter (similar feel, open source)

## UI Vibe
- Clean, corporate, modern
- Lots of whitespace
- Card-based layout
- Subtle shadows, rounded corners
- Professional but not stuffy
